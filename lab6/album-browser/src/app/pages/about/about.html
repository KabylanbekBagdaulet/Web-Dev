About

Project

Album Browser is a multi-page SPA built with Angular routing, HttpClient, and Observables — Lab 6 of the Web Application Development course at KBTU.

Data is fetched from JSONPlaceholder. Photos are sourced from Picsum Photos. All CRUD operations update the UI optimistically.
